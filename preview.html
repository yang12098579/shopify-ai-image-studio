How It Works

1. Install the app from Shopify App Store
2. Connect your Pollinations.ai API key
3. Select a product & describe the scene
4. Generate — AI creates a pro product photo
5. Download & upload to your product listing

Tech Stack

Framework: Shopify Remix + React
UI: Polaris Design System
AI: Pollinations.ai (Free) API
Database: Prisma + SQLite
Auth: Shopify OAuth 2.0
Deployment: Shopify Cloud / Vercel
